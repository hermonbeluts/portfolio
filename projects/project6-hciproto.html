Travel App Prototype

Enhancing Travel Experiences through Authentic Connections | Spring 2023

Project Overview

As part of a collaborative effort in a human-computer interaction course, my team conceptualized and developed a prototype for a travel app designed to connect travelers with locals for personalized recommendations. Our app focuses on enhancing the travel experience by bridging the gap between tourists and local communities, fostering authentic and meaningful connections.

Development Process

The project followed a rigorous UX design process that included:

  • Idea Generation: Brainstorming unique travel-focused concepts to identify unmet user needs.
  • User Research: Creating user personas to understand potential user requirements and expectations.
  • Prototyping: Designing physical prototypes to explore features like user profiles, local matching, and chat interfaces.
  • User Testing: Conducting iterative rounds of usability testing and interviews to refine the app's functionality and design.
  • Interactive Prototype: Developing the final app prototype on Figma with interactive features like personalized profiles, local screening, and secure chatting.

Skills and Technologies

This project highlights my proficiency in:

  • User Experience Design: Developing intuitive user flows and engaging interface designs.
  • Prototyping Tools: Leveraging Figma to create interactive prototypes.
  • Research Ethics: IRB-certified for conducting user research and interviews.
