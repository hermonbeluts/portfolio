Apple Festival Website

Redefining Local Community Engagement | Fall 2022

Project Overview

The Apple Festival is a celebrated annual event in my college town, bringing together local vendors, performers, and attendees. This project, part of a programming course, aimed to design a visually appealing, user-friendly website that consolidates festival details, making them accessible to a broad audience.

Development Process

I began by researching various sources to gather comprehensive details about the festival, including dates, times, locations, vendor information, and performance schedules. I then structured this information into a cohesive layout that balanced design and functionality. The development process involved:

  • Sketching and iterating on wireframes for optimal user navigation.
  • Developing the website using HTML, CSS, and JavaScript for interactivity.
  • Testing for responsiveness and usability across multiple devices.

Skills and Technologies

This project highlights my expertise in:

  • Web Development: Proficient use of HTML, CSS, and JavaScript to create a responsive and interactive website.
  • Design Iteration: Conducting multiple iterations to refine user experience.
  • Tools: Leveraging VSCode for efficient development and debugging.

Final Product

The website features an intuitive navigation structure, visually appealing design, and rich content about the festival. Code is hosted locally (available upon request). You can view the final output here.
