Trending TikTok Audio Analysis

Discover Patterns Behind Trending TikTok songs | Fall 2023

Project Overview

This project aimed to analyze the evolution of trends on TikTok, identify patterns in trending songs, and understand how songs gain or lose popularity. Using a Kaggle dataset collected via the Spotify API, we examined various features like artist popularity, danceability, and loudness to determine their significance or effect on a song's popularity or "trendiness".

Process and Findings

We analyzed a Kaggle dataset of popular songs on TikTok from 2019-2022, collected using the Spotify API and containing several features related to song popularity. We ultimately cleaned the dataset to observe columns/features track name and popularity, artist name and popularity, track popularity, year, danceability, energy, valence, loudness, and duration. Additionally, we merged the separate datasets for each year into one consolidated dataset and proceeded to further clean the data for our analysis. We then preregistered our hypothesis and conducted our analysis of which factors influenced a song’s trendiness using various data science libraries in Python such as NumPy and Pandas, as well as SQL. We generated several visualizations to look for trends between the features and popularity. We ran ordinary least squares (OLS) regression analysis on these features to determine if there was any correlation. Key findings included:

  • An increase in a song’s loudness slightly correlates with its popularity.
  • Energy and valence do not significantly impact trendiness.
  • Danceability, energy, loudness, and valence increased from 2019 to 2022.
  • Artist popularity is a strong predictor of song trendiness, with top artists producing about 25% of trending songs.
