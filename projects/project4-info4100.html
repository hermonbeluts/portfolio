Clicker Data Dashboard

Interactive learning analytics dashboards for students and instructors.

Project Overview

This project focused on developing interactive dashboards for a Cornell CS course to visualize clicker and quiz data. Using RShiny, we created two dashboards: an instructor dashboard to monitor class-wide performance and engagement, and a student dashboard to provide personalized insights into individual performance.

Datasets

We worked with two datasets:

  • Experience Dataset: Contains aggregated data from surveys and faculty records, such as programming experience, SQL knowledge, and data visualization skills.
  • Quiz Dataset: Includes iClicker and quiz data, with attributes such as attendance, correct answers, and quiz scores.

Process

Our work involved the following steps:

  • Planning: Sketched wireframes for both dashboards, identifying user needs and visualization goals.
  • Wireframe Implementation: Built layouts for the dashboards using RShiny.
  • Data Pre-processing: Cleaned and merged datasets to prepare them for visualization.
  • Visualization: Created data visualizations for both dashboards to display key metrics.

Dashboards

  • Instructor Dashboard: Displays metrics such as total students, average attendance, distribution of skills, and quiz performance.
  • Student Dashboard: Displays individual quiz scores, attendance, session temperatures, and comparative cohort performance.

Personal Contribution

My primary responsibilities included planning and sketching the dashboards and creating data visualizations for the instructor dashboard, particularly the visualization of quiz performance by program.

Skills Demonstrated

  • Proficiency in R, RStudio, and RShiny for dashboard development.
  • Data cleaning, analysis, and visualization techniques.
  • Collaboration in a team environment to produce a cohesive project.
